# Agent Performance

Historical scoring updated after each task.
An agent with a recurring average score <= 2 is banned from the corresponding task type. See `project-context.md`.

## Rating Scale

| Score | Meaning |
| --- | --- |
| 5 | Exceptional - exceeds expectations, unique insights |
| 4 | Good - acceptance criteria met, solid quality |
| 3 | Satisfactory - acceptable work, some minor gaps |
| 2 | Insufficient - missed criteria, retry required |
| 1 | Failure - unusable output, escalation required |
